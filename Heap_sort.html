힙 정렬(Heap Sort)

참고사이트 : 동빈나-실전 알고리즘 강좌-힙정렬(Heap Sort)

  • 힙 트리 구조(Heap Tree Structure)를 이용하는 정렬 방법
  • 이진 트리 : 모든 노드의 자식 노드가 2개 이하인 트리 구조
  • 완전 이진 트리 : 데이터가 루트 노드부터 시작해서 자식 노드가 왼쪽 자식 노드, 오른쪽 자식노드로 차근차근 들어가는 구조
  • 힙(Heap) : 최솟값이나 최댓값을 빠르게 찾아내기 위해 완전 이진 트리를 기반으로 하는 트리
  • 최대 힙 : '부모 노드'가 '자식 노드'보다 큰 합
  • 힙 정렬을 하기 위해서는 정해진 데이터를 힙 구조를 가지도록 만들어야한다.
  • 힙 생성 알고리즘(Heapify Algorithm) : 특정한 하나의 노드에 대해서 수행한다.
    하나의 노드를 제외하고는 최대 합이 구성되어 있는 상태라고 가정한다.
    특정한 노드의 두 자식 중에서 더 큰 자식과 자신의 위치를 바꾸는 알고리즘이다.
    위치를 바꾼 이후에 여전히 자식이 존재하는 경우 또 자식 중에서 더 큰 자식과 자신의 위치를 바꾸어야한다.
  • 힙 생성 알고리즘이 실행되는 시간은 높이와 동일 => log2N
  • 힙을 만드는 시간 복잡도는 O(n)이다.
  • 힙 정렬의 전체 시간 복잡도는 O(n*log N)이다. (항상 보장한다.)
  • 별도의 부가적인 배열이 필요하지 않다는 점에서 메모리 측면에서 효율적이다.

#include<stdio.h>
 
int number = 9;
int heap[9] = { 7,6,5,8,3,5,9,1,6 };
 
int main(void) {
    // 먼저 전체 트리 구조를 최대 힙 구조로 바꿉니다.
    for (int i = 1; i < number; i++) { // 
        int c = i;
        do {
            int root = (c - 1) / 2; // 자기 자신의 부모 노드
            if (heap[root] < heap[c]) { // 부모의 값보다 자식의 값이 크다면
                int temp = heap[root];
                heap[root] = heap[c];
                heap[c] = temp; // 위치를 바꾼다.
            }
            c = root; // 자기 부모 노드로 이동해서 반복적으로 수행한다.
 
        } while (c != 0); 
    }
    //크기를 줄여가며 반복적으로 힙을 구성
    for (int i = number - 1; i >= 0; i--) { 
        int temp = heap[0];
        heap[0] = heap[i];
        heap[i] = temp;// root 노드와 가장 마지막에 있는 노드를 바꾼다.
 
        int root = 0;
        int c = 1;
        do { //heap 구성
            c = 2 * root + 1;
            // 자식 중에 더 큰 값을 찾는다.
            if (heap[c] < heap[c + 1] && c < i - 1) {
                c++; // 오른쪽으로 이동한다. => 더 큰 값이 c 에 담기도록 한다.
 
            }
            //루트보다 자식이 더 크다면 교환
            if (heap[root] < heap[c] && c < i) {
                int temp = heap[root];
                heap[root] = heap[c];
                heap[c] = temp;
 
            }
            root = c; // 바꾼 후에는  c를 root로 이동시켜서 재귀적으로 heap 구조를 만들어준다.
        } while (c < i);
 
    }
 
    for (int i = 0; i < number; i++) {
        printf("%d ", heap[i]);
    }
}
